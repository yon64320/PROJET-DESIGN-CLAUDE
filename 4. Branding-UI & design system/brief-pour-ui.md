## Contexte projet

- Site de l'agence elle-même (pas d'un client final)
- Agence : accompagnement premium de digitalisation pour PME — site web orienté conversion + automatisation + IA
- Aucune identité visuelle existante confirmée (logo à créer / à venir)
- Structure jeune / en lancement
- Ton : premium, moderne, clair, crédible, intelligent — pas technique, pas "agence marketing cliché"
- Aucune preuve chiffrée disponible : la crédibilité repose sur le process et la qualité perçue
- Pricing non affiché — CTA = appel découverte

## Contraintes de marque

- Niveau de sobriété : **premium discret** — pas de flashy, pas de minimalisme vide
- Pas de design générique type "agence digitale bleue avec dégradé violet"
- Le site lui-même est la démonstration de l'offre — il doit incarner "ce que l'agence peut faire pour vous"
- Mobile-first obligatoire
- Performance : éviter les effets coûteux sans valeur business

## Pages à couvrir

Home, Offre, Contact/RDV, À propos + pages légales

## Arbitrages UX déjà validés

- Navigation sticky avec CTA permanent "Réserver mon appel découverte"
- Max 4 liens nav + 1 CTA
- Réalisations masquées en v1 — remplacées par section process renforcée sur Home
- CTA secondaire = audit de l'existant (formulaire 2 champs)
- Formulaire Contact : Calendly en priorité + fallback 3 champs
